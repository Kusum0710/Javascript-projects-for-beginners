# JavaScript Learning Projects

Welcome! This repository is a collection of projects I built while beginning my journey in JavaScript. If you're starting your own JavaScript learning journey, I hope these projects will help you understand fundamental concepts and best practices.

## 🎯 Purpose

This repo showcases beginner-friendly JavaScript projects that demonstrate:
- DOM manipulation and event handling
- LocalStorage and data persistence
- HTML form validation
- CSS styling and animations
- JavaScript ES6+ features (arrow functions, template literals, etc.)

## 📁 Projects

### Quiz App
A cute, fun and interactive quiz application to test your knowledge.

**Features:**
- Multiple-choice questions with instant feedback
- Score tracking and calculation
- Progress indicator showing current question number
- Shuffle questions for variety
- Immediate visual feedback (correct/incorrect)
- Final score display with performance summary
- Clean and intuitive UI

**Technologies Used:** HTML5, CSS3, Vanilla JavaScript

**How to Use:**
1. Open `Quiz App/index.html` in your browser
2. Read each question carefully and select your answer
3. Click "Next" to proceed to the next question
4. View your final score at the end of the quiz
5. Retake the quiz anytime to improve your score

---

### Expense Tracker
A simple yet functional expense tracker application that helps you manage your income and expenses.

**Features:**
- Add and delete transactions
- Automatic balance calculation
- Separate income and expense tracking
- Data persistence using LocalStorage
- Currency support (Indian Rupee)
- Responsive design with smooth animations

**Technologies Used:** HTML5, CSS3, Vanilla JavaScript

**How to Use:**
1. Open `Expense Tracker/index.html` in your browser
2. Enter a transaction description and amount
3. Use negative numbers for expenses, positive for income
4. Transactions are saved automatically to your browser's LocalStorage

## 🚀 Getting Started

No build tools or installation required! All projects are built with vanilla HTML, CSS, and JavaScript.

Simply open the `index.html` file in any modern web browser to run the projects.

## 💡 What You'll Learn

- **HTML**: Semantic structure and form elements
- **CSS**: Layouts, flexbox/grid, animations, and gradients
- **JavaScript**: Event listeners, DOM manipulation, LocalStorage, array methods (map, filter, reduce)

## 📝 Tips for Beginners

- Read through the code comments to understand the logic
- Experiment with the code—try modifying styles or adding new features
- Use browser DevTools (F12) to debug and inspect elements
- Start small and build on top of these projects
